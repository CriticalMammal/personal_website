Project Rolodex

Source Code

This project started because I wanted to create the 3d "layered" look it currently has. It's also where the name "Rolodex" came from, since the visuals sort of resemble how a rolodex functions.

It's programmed in C++ with the Simple Direct Media Layer (SDL) library. Midway through development I was planning on turning it into some sort of puzzle game, though those mechanics never made it in because quite a lot of work was spent creating the backbone and core functionality.
